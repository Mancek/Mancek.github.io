CSS

CSS je jezik stiliziranja koji služi za opisivanje izgled dokumenta napisanog na jeziku kao što je HTML. CSS je temeljna tehnologija World Wide Weba, zajedno s HTML-om i JavaScript-om.

CSS je dizajniran tako da omogući odvajanje prezentacije i sadržaja, uključujući izgled, boje i fontove. Ovo odvajanje može poboljšati dostupnost sadržaja, pružiti veću fleksibilnost i kontrolu u specificiranju karakteristika prezentacije, omogućiti više web stranica za dijeljenje oblikovanja specificiranjem odgovarajućeg CSS-a u zasebnoj .css datoteci i smanjiti složenost i ponavljanje strukturnog sadržaja.

Sada ćemo objasnit neka svojstva CSS-a koja su prisutna samo u novijim verzijama, te koje možda još nisu podržane u najnovijim verzijama web preglednika.

CSS animacije

CSS animacije dopuštaju animaciju HTML elemenata bez korištenja JavaScript-a.

Animacije!

Naravno, animacije nisu podržane na starijim preglednicima. Animacije se mogu stvarati na dva načina. Prvi način je uz pomoć transition CSS svojstva. Sljedeći način stvaranja animacija je uz pomoć @keyframe pravila. Ako bi koristili @keyframe za gore navedeni primjer, to bi izgledalo ovako: @keyframes animation-demo {
  from { background-color: red; }
  to { background-color: green; }
}
.animation-demo {
  padding: 5px;
  animation: animation-demo 4s alternate ease-in-out infinite;
}

CSS kalkukacije

CSS kalkulacije nam dopuštaju kalkulaciju jednostavnijih jednadžba uz pomoć calc() funkcije.

Primjer kalkulacije
width: calc(10vw + 30px + 10rem);
Jedno važno svojstvo calc() funkcije je da nam dopušta zbrajanje različitih jedinica za veličinu. Tako smo npr. u ovom primjeru zbrajali vw jedinice, px jedinice i rem jedinice.

CSS gradijenti

Gradijenti nam omogućuju stvaranje svakavih slika koje se većinom koriste kao pozadine na web stranicama.

Ima puno vrsta gradijenta, u prva dva primjera korišten je linear-gradient(), dok je u trećem primjeru korišten radial-gradient(). Ove funkcije primaju puno različitih parametra, te je potencijal s njima ogroman. Za generiranje gradijenta često se koriste generatori poput ovog.

Fontovi

Možda je teško za vjerovati, ali prije samo nekoliko godina, postojala je samo nekolicina fontova koji su se mogli koristiti na web-stranicama. Sada imamo pomagala poput Google Fonts koji nam dopuštaju dodavanje fontova na web stranice bez ikakvih problema.

Fontovi se mogu uvesti u web stranice na nekoliko načina. Može se koristiti <link /> element u HTML-u, ili pak uz pomoć @import pravila.

U HTML-u: <link href="link-fonta" rel="stylesheet">
U CSS-u: @import url('link-fonta');
Nakon što uvezemo neki font, možemo ga koristiti u CSS-u uz pomoć font-family pravila.
.font-1 {
  font-family: "Ime Fonta";
}

Media Queries

Jedno od najvažnijih dodataka u novijim verzijama CSS-a su media queries. Oni nam pružaju mnogo mogućnosti, ali najvažnija je mogućnost drugačiji prikaz stranice za mobitele, tablete i računala, ovisno o veličini ekrana.

Računalo

Mobitel

Na primjer možemo mijenjati boju ove kocke ovisno o tome je li stranica otvorena na mobitelu ili na računalu:

Kocka

Sada će pozadina ove kocke biti zelena na računalu sa bijelim tekstom, dok će se na mobitelu i ostalim manjim ekranima prikazivati kao žuta kocka sa crnim tekstom.

To smo postigli ovako:
dali smo kocki pozadinsku boju i boju teksta uz pomoć običnog CSS-a
.media-query-example {
  background-color: green;
  color: white;
}
Zatim smo napravili media query pravilo koje će promijeniti pozadinsku boju kada je ekran manji od 768px:
@media only screen and (max-width: 768px) {
  .media-query-example {
    background-color: yellow;
    color: black;
  }
}
